# Sprint_4
Проект содержит автотесты для учебного сервиса [Яндекс.Самокат](https://qa-scooter.praktikum-services.ru). 

## Тестовые сценарии
1. Выпадающий список в разделе «Вопросы о важном» содержит соответствующий ответ на вопрос.
2. Заказ самоката. Проверка позитивного сценария с двумя наборами данных.
Из чего состоит позитивный сценарий:
   - Нажать кнопку «Заказать». На странице две кнопки заказа.
   - Заполнить форму заказа.
   - Проверить, что появилось всплывающее окно с сообщением об успешном создании заказа.
3. Проверка двух точек входа в "сценарий" заказа.

## Технологии, использующиеся в проекте
1. Java 11
2. Maven 3.9.9
3. JUnit 4.13.2
4. Selenium-Java 4.17.0
5. WebDriverManager 4.4.3

## Запуск тестов
Чтобы запустить тесты, выполните следующую команду в терминале:
```bash
mvn clean compile
mvn test
```

## Подключение Maven и WebDriver для запуска тестового проекта

1. Установка Maven
   Убедитесь, что Maven установлен на вашем компьютере. Если нет, выполните следующие шаги:

   - Скачайте Maven с [официального сайта](https://maven.apache.org/download.cgi).
 
   - Распакуйте архив и настройте переменную окружения PATH, добавив путь к папке bin Maven.

   - Проверьте установку, выполнив в терминале:

```bash
mvn -v
```

2. Настройка WebDriver
   - Скачайте драйвер для вашего браузера (например, [ChromeDriver](https://sites.google.com/chromium.org/driver/)). 
   - Укажите путь к драйверу в коде или добавьте его в переменные окружения.
